## 一、混沌理论基础（4 个模型）

### 1.1 逻辑斯蒂映射——给每个社会变量一个 r 值

**公式**
```
x_{n+1} = r · x_n · (1 - x_n)
```

**r 值决定系统行为**

| r 范围 | 动力学状态 | 小说意义 |
|--------|-----------|---------|
| r < 1 | 灭绝 | 低于阈值的社群注定消亡 |
| 1 < r < 3 | 稳定均衡 | 可生存但僵化、可预测 |
| 3 < r < 3.57 | 周期振荡 | 繁荣-萧条循环 |
| r > 3.57 | **混沌** | 永不重复、对初值极度敏感 |

**核心思想**：把社会系统每个关键变量（信任、偏执、合作意愿）视为一个逻辑斯蒂映射。每个映射的 r 值由事件动态更新。

```rust
pub struct LogisticMap {
    pub r: f64,         // 控制参数
    pub x: f64,         // 当前状态 [0, 1]
    pub history: VecDeque<f64>,
}

impl LogisticMap {
    pub fn step(&mut self) -> f64 {
        self.x = self.r * self.x * (1.0 - self.x);
        self.history.push_back(self.x);
        self.x
    }

    pub fn regime(&self) -> Regime {
        match self.r {
            r if r < 1.0 => Regime::Extinction,
            r if r < 3.0 => Regime::Stable,
            r if r < 3.5699 => Regime::Periodic,
            _ => Regime::Chaotic,
        }
    }
}
```

**应用**：一次背叛事件把 group_trust.r 从 2.8 推到 3.5，信任从稳定变成混沌振荡。

**论文**：May (1976), *Nature*, 261, 459-467; Feigenbaum (1978), *J. Stat. Phys.*, 19(1), 25-52

### 1.2 李雅普诺夫指数（蝴蝶效应量化）

**公式**
```
λ = lim_{N→∞} (1/N) · Σ_{n=0}^{N-1} ln|f'(x_n)|

对逻辑斯蒂映射：f'(x) = r·(1 - 2x)
```
- λ > 0：混沌，初始条件指数发散
- λ = 0：周期/准周期
- λ < 0：稳定

**蝴蝶效应验证方案**：克隆引擎，改 0.1% 初始条件，跑 1000 tick。若两个模拟的状态距离 > 0.5，说明有正 Lyapunov 指数。

**论文**：Lorenz (1963), *J. Atmos. Sci.*, 20(2), 130-141; Benettin et al. (1980), *Meccanica*, 15, 9-20

---

### 1.3 奇异吸引子——社会三变量模型

**公式**（类洛伦兹系统，变量重映射为社会学意义）
```
dx/dt = σ(y - x)    // 群体情绪（团结/分裂）
dy/dt = x(ρ - z) - y  // 资源压力（充裕/匮乏）
dz/dt = xy - βz      // 制度控制（权威严格性）

参数：σ=10, ρ=28, β=8/3（经典混沌值）
```

**特性**：系统在有限区域内永不重复地振荡、有界但不可预测、对初值敏感。

**论文**：Ruelle & Takens (1971), *Comm. Math. Phys.*, 20(3), 167-192

---

### 1.4 分岔与临界减速（崩溃前兆）

**公式**
```
鞍结分岔：dx/dt = r + x²
叉形分岔：dx/dt = rx - x³
霍普夫分岔：稳定不动点→极限环

临界减速：分岔点前恢复率 → 0
```

**可计算的社会崩溃预警**：每次扰动后测量信任/群体士气的恢复率。当恢复率 < 0.01 时：系统即将分支，社会可能崩溃。

**论文**：Strogatz (2018), *Nonlinear Dynamics and Chaos*; Scheffer et al. (2009), *Nature*, 461, 53-59

## 二、自组织临界性（3 个模型）

### 2.1 SOC 沙堆模型（Bak, Tang & Wiesenfeld, 1987）

**规则**
```
1. 随机落下一粒沙（压力 +1）
2. 任意格点高度 ≥ 4 → 崩塌：向 4 个邻居各传 1 粒
3. 邻居可能连锁崩塌
4. 事件规模 = 一次级联中崩塌的格点数
```

**关键发现**：系统在没有外部调节的情况下自组织到临界状态。崩塌规模服从幂律分布。

```rust
pub struct CrisisSandpile {
    pub stress: HashMap<Uuid, f64>,
    pub threshold: f64,       // 默认 4.0
    pub avalanche_sizes: Vec<u64>,
}

impl CrisisSandpile {
    pub fn add_stress(&mut self, char_id: Uuid, amount: f64) -> u64 {
        let stress = self.stress.entry(char_id).or_insert(0.0);
        *stress += amount;
        if *stress >= self.threshold {
            *stress = 0.0;
            self.cascade(char_id)
        } else { 0 }
    }

    fn cascade(&mut self, origin: Uuid) -> u64 {
        let mut queue = VecDeque::new();
        queue.push_back(origin);
        let mut avalanche = 0;
        while let Some(current) = queue.pop_front() {
            avalanche += 1;
            for neighbor in /* relationship graph neighbors */ {
                let s = self.stress.entry(neighbor).or_insert(0.0);
                *s += 0.5;
                if *s >= self.threshold { *s = 0.0; queue.push_back(neighbor); }
            }
        }
        self.avalanche_sizes.push(avalanche);
        avalanche
    }
}
```

**论文**：Bak, Tang & Wiesenfeld (1987), *PRL*, 59(4), 381-384; Bak (1996), *How Nature Works*

### 2.2 幂律分布——社会系统在极端国度运行

**公式**
```
P(size) ~ size^(-α)
其中 α ~ 1.0-2.5，社会系统 α ~ 1.5
```

**不要用高斯分布**：社会系统产生极端事件的频率远高于正态预测。

```rust
fn sample_power_law(alpha: f64, x_min: f64, rng: &mut impl Rng) -> f64 {
    let u: f64 = rng.gen();
    x_min * (1.0 - u).powf(-1.0 / (alpha - 1.0))
}
```

**中值国度 vs 极端国度**

| 性质 | 中值国度(高斯) | 极端国度(幂律) |
|------|-------------|-------------|
| 最大值 | ~均值的 3 倍 | 可以大 100 倍以上 |
| 是否可预测 | 是 | 否 |
| 示例 | 身高、体重 | 战争、金融危机、丧尸爆发 |

**论文**：Mandelbrot (1963), *J. Business*, 36(4), 394-419; Taleb (2007), *The Black Swan*; Clauset et al. (2009), *SIAM Review*, 51(4), 661-703

---

### 2.3 Lotka-Volterra 三物种振荡

**公式**
```
dR/dt  = αR - βRS       // 资源（自再生，被幸存者消耗）
dS/dt  = δRS - γS - εSZ // 幸存者（被资源驱动，被丧尸杀死）
dZ/dt  = ζSZ - ηZ       // 丧尸（从杀死幸存者获得增长）
```

三物种系统可产生混沌动力学（两物种只有周期振荡）。

**论文**：Lotka (1925); Volterra (1926); Hastings & Powell (1991), *Ecology*, 72(3), 896-903

## 三、复杂适应系统（4 个模型）

### 3.1 CAS 四属性（Holland, 1995）

| 属性 | 机制 | 引擎实现 |
|------|------|---------|
| 聚合 | 个体局部规则 → 宏观结构涌现 | 派系通过社区检测自动发现 |
| 标记 | 使用标签筛选交互对象 | 动态标签系统（14 号文档） |
| 非线性 | 微小变化可产生巨大效应 | 逻辑斯蒂混沌 + SOC 级联 |
| 流 | 资源/信息/影响力在节点间流动 | 资源级联 + 社交涟漪 |

**论文**：Holland (1995), *Hidden Order*

### 3.2 CAS 三机制（Gell-Mann, 1994）

| 机制 | 实现 |
|------|------|
| 多样性 | 10 维人格差异 + 技能差异 + 背景差异 |
| 内部模型 | 角色的信念系统（belief_trait_pressure） |
| 构建模块 | 角色/物品/任务/协议作为基本构件 |

**论文**：Gell-Mann (1994), *The Quark and the Jaguar*

### 3.3 边缘的混沌——Langton λ 参数

```
λ = 0 → 完全死板（所有决策可预测）
λ = 0.3-0.4 → 边缘的混沌（既有秩序又有新奇）← 目标区间
λ = 1.0 → 完全随机（无叙事连贯性）
```

**动态调谐**：系统太稳定（所有角色长期舒适）→ λ 增加；系统太混乱（死亡事件频发）→ λ 减少。

**论文**：Langton (1990), *Physica D*, 42(1-3), 12-37

### 3.4 路径锁入与增加回报（Arthur, 1989）

**公式**
```
P(choose A) = perceived_A / (perceived_A + perceived_B)
perceived_A = fitness_A + adoption_count · network_effect_strength
```

早期随机选择因增加回报（信任飞轮、资源复利）而被锁定。

**论文**：Arthur (1989), *EJ*, 99(394), 116-131; David (1985), *AER*, 75(2), 332-337

## 四、级联效应（4 个模型）

### 4.1 Watts 阈值级联（2002）

```
P(adopt | neighbor) = fraction_adopted_neighbors ≥ threshold_i
级联规模服从幂律——大部分很小，小部分覆盖全网
```

**论文**：Watts (2002), *PNAS*, 99(9), 5766-5771

### 4.2 负载分布级联（Motter & Lai, 2002）

```
节点失效 → 负载均分给邻居 → 邻居可能超载失效 → 连锁反应
```

### 4.3 二阶/三阶效应引擎

```
一阶（100%，即时）：直接后果
二阶（75%，1-24h）：直接相关方的反应
三阶（30%，1-7d）：目击者的适应行为
四阶+（10%，7-30d）：间接因果链
```

### 4.4 眼镜蛇效应（反直觉的政策后果）

```
当指标成为目标 → 该指标不再是好指标（Goodhart's Law）
人们会"优化"指标而非实际目标
```

## 五、强韧性与反脆弱（2 个模型）

### 5.1 脆性/韧性/反脆弱（Taleb, 2012）

```
脆性：受冲击受损（瓷器）
韧性：不受冲击影响（钢球）
反脆弱：因冲击受益（免疫系统）
```

**群体反脆弱性**
```rust
fn group_antifragility(group: &Faction) -> f64 {
    let diversity = skill_diversity(group);
    let redundancy = role_redundancy(group);
    let modularity = social_modularity(group);  // 模因度
    let learning = learning_capacity(group);
    (diversity + redundancy + modularity + learning) / 4.0
}
```

**论文**：Taleb (2012), *Antifragile*

### 5.2 压力接种（Meichenbaum, 1985）

```
适度压力：产生抵抗力 → 抗脆弱
过度压力：造成创伤 → 脆弱
最佳压力：过去总压力的 30-60% → 产生最大成长
```
