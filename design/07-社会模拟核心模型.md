## 一、意见动力学（6 个模型）

### 1.1 DeGroot 线性共识模型（1974）

**公式**
```
x_i(t+1) = Σ_j w_ij · x_j(t)

其中 Σ_j w_ij = 1（行归一化）
```

**变量**
- `x_i(t)`：角色 i 在时间 t 的意见值 [0,1]
- `w_ij`：角色 i 赋予角色 j 的意见权重（从信任度映射）
- `w_ii`：自我权重（1 - 其他人权重之和）

**适用场景**：角色之间理性讨论某话题，最终意见收敛到加权平均。

**Rust 实现**
```rust
fn degroot_step(opinions: &[f64], weights: &[Vec<f64>]) -> Vec<f64> {
    let n = opinions.len();
    let mut next = vec![0.0; n];
    for i in 0..n {
        for j in 0..n {
            next[i] += weights[i][j] * opinions[j];
        }
    }
    next
}

fn build_trust_weights(relationships: &[Relationship], n: usize) -> Vec<Vec<f64>> {
    let mut w = vec![vec![0.0; n]; n];
    for i in 0..n {
        let mut total = 0.0;
        for j in 0..n {
            if i != j {
                let t = relationships[i].trust_with(j) as f64;
                if t > 0.0 { w[i][j] = t / 10.0; total += w[i][j]; }
            }
        }
        w[i][i] = 1.0 - total.min(1.0);
    }
    w
}
```

**关键性质**：若社交网络强连通，意见收敛到加权平均值。收敛值 = Σ π_i · x_i(0)，其中 π 是权重矩阵的主左特征向量。

**论文**：DeGroot (1974), *JASA*, 69(345), 118-121

### 1.2 Friedkin-Johnsen 锚定模型（1999）

**公式**
```
x_i(t+1) = λ_i · x_i(0) + (1 - λ_i) · Σ_j w_ij · x_j(t)
```

**变量**
- `x_i(0)`：初始意见（锚定值）
- `λ_i ∈ [0,1]`：固执度。λ=0 完全跟随，λ=1 完全坚持初始

**从人格计算固执度**
```rust
fn lambda_from_personality(p: &PersonalityTraits) -> f64 {
    // 高宜人=易受影响，高神经质=固守己见，高开放=接受新观点，高冷酷=不在乎他人
    let raw = (1.0 - p.agreeableness * 0.3)
            + (p.neuroticism * 0.25)
            + (p.ruthlessness * 0.25)
            - (p.openness * 0.2);
    raw.clamp(0.1, 0.95)
}
```

**适用场景**：角色对某议题有初始立场，讨论后立场微调但不会完全放弃。**这是最实用的小说意见模型**。

**论文**：Friedkin & Johnsen (1999), *Advances in Group Processes*, 16, 1-29

---

### 1.3 Hegselmann-Krause 有界信任模型（2002）

**公式**
```
x_i(t+1) = (1 / |N_i(t)|) · Σ_{j ∈ N_i(t)} x_j(t)
其中 N_i(t) = { j : |x_i(t) - x_j(t)| < ε }
```

**参数 ε 决定群体分裂程度**

| ε 值 | 行为 | 小说含义 |
|------|------|---------|
| ε > 0.5 | 全局共识 | 所有人都能沟通 |
| 0.2 < ε < 0.5 | 多派系 | 形成 2-3 个意见集群 |
| ε < 0.2 | 碎片化 | 每人固守己见 |

**适用场景**：当角色间的意见分歧过大时，他们互相无视而不是互相说服——这是派系形成的数学源头。

**论文**：Hegselmann & Krause (2002), *JASSS*, 5(3)

---

### 1.4 信息级联模型（Bikhchandani, 1992）

**公式**
```
级联触发条件：|#Accept - #Reject| ≥ 2
触发后：所有人忽略自己的私人信号，跟随多数
```

**适用场景**：基于不完整信息的灾难性集体决策——一个错误共识因为"看起来大家都这么觉得"而锁定。

**论文**：Bikhchandani et al. (1992), *JPE*, 100(5), 992-1026

---

### 1.5 群体极化模型（Isenberg, 1986）

**公式**
```
x_i(t+1) = x_i(t) + α · (mean_of_similar_opinions(t) - x_i(t))
其中 "similar_opinions" = {j: x_j(t) 与 x_i(t) 在同一侧}
```

**适用场景**：讨论后群体变得更加极端——支持严厉惩罚的人在讨论后变得更加严厉。

**论文**：Isenberg (1986), *JPSP*, 50(6), 1141-1151

---

### 1.6 社会比较理论（Sanders & Baron, 1977）

**公式**
```
x_i(t+1) = x_i(t) + β·(x_group_mode - x_i(t)) + γ·(x_desired - x_i(t))
其中 β = 从众强度，γ = 差异强度
```

**高宜人角色**：β 高 γ 低 → 向均值靠拢
**低宜人角色**：β 低 γ 高 → 走向极端以维持独特性

**论文**：Sanders & Baron (1977), *JESP*, 13(4), 303-314

## 二、社会规范（3 个模型）

### 2.1 Bicchieri 规范激活模型（2006）

**公式**
```
规范 N 激活当且仅当：
  (1) E_i(N) ≥ θ_E    // 经验期望：足够多人遵守
  (2) N_i(N) ≥ θ_N    // 规范期望：足够多人认为我应该遵守
  (3) P_i(N) > Cost_i(N) // 偏好 > 成本
```

**违反决策**
```
U_i(violate) = MaterialGain - PunishmentRisk·Severity - InternalGuilt - SocialDisapproval
U_i(comply) = 0
违反当 U_i(violate) > U_i(comply)
```

**适用场景**：规范为什么会崩溃——当越来越少人遵守（E_i 下降），且惩罚失效（PunishmentRisk↓），违反成为理性选择。

**论文**：Bicchieri (2006), *The Grammar of Society*, Cambridge

---

### 2.2 规范雪崩临界点（Granovetter, 1978）

**公式**
```
设 f(t) = 遵守规范的人口比例
规范崩溃发生在 f(t) < (I - R) / (S - R)
其中 I = 内在动机，R = 违反回报，S = 社会压力
```

**适用场景**：饥饿加剧 → R 上升 → 临界点上移 → 更容易崩塌。一旦 f 低于临界点，剩余遵守者认为"没人遵守了" → 雪崩。

**论文**：Granovetter (1978), *AJS*, 83(6), 1420-1443

---

### 2.3 惩罚执行决策

**公式**
```
Punish_i 当 Benefit > Cost
Benefit = DeterrenceEffect_i + ReputationGain_i
Cost = DetectionCost + ActionCost + RetaliationRisk
```

**论文**：Fehr & Gachter (2002), *Nature*, 415, 137-140

## 三、资源共享与分配（5 个模型）

### 3.1 需求权重分配

```
x_i = (need_i / Σ need_j) · total
need_i = w₁·hunger_i + w₂·health_i + w₃·age_i + w₄·labor_i
```

### 3.2 混合分配模型

```
x_i = α·(1/n) + β·(need/Σneed) + γ·(contribution/Σcontribution)
其中 α+β+γ = 1，参数随稀缺度自动调整
```

**稀缺度决定参数**
```
scarcity_ratio = total / (survival_threshold · n)
scarcity_ratio < 0.8  → α=0, β=1, γ=0    // 纯按需
scarcity_ratio > 1.2  → α=0.3, β=0.2, γ=0.5 // 更重贡献
```

### 3.3 罗宾汉再分配

```
再分配触发条件：Gini > 0.3 - 0.1·cohesion - 0.15·threat + 0.2·individualism
```

### 3.4 稀缺心态与认知税（Mullainathan & Shafir, 2013）

```
可用带宽 = W_max - k · scarcity^p   (p ≈ 1.5-2.0)
决策质量 = Q_max · (带宽/W_max)^γ   (γ ≈ 1.5-2.0)
隧道视野激活：scarcity > 0.6-0.7
```

**适用场景**：第 195 章吃尸体事件——不是角色变蠢了，是稀缺把认知带宽烧光了。带宽降到 50% 时决策质量降到约 25%。

**论文**：Mullainathan & Shafir (2013), *Scarcity*

### 3.5 享乐适应与伊斯特林悖论

```
情绪偏离基线按指数衰减回基线
适应半衰期：正面事件 ~3 天，负面事件 ~7-30 天
高神经质角色适应正面快、适应负面慢
```

## 四、互惠模型（3 个模型）

### 4.1 直接互惠：以牙还牙（Axelrod, 1984）

```
cooperate_t = (t == 0) ? true : opponent_cooperated_{t-1}
```

**重复博弈稳定合作条件**
```
w > (T - R) / (T - P)
w = 再次相遇概率
在小群体中 w → 1.0 → 合作稳定
```

**论文**：Axelrod (1984), *The Evolution of Cooperation*

### 4.2 间接互惠：形象评分（Nowak & Sigmund, 1998）

```
I_i += α·(benefit - cost)  // 帮助
I_i -= β·need               // 拒绝帮助（仅当 need > 冷漠阈值）
P(help) = σ(γ · (I_i - reputation_threshold))

声誉传播衰减：I_know_i = I_i · (1 - distance_ik / max_distance)
```

**论文**：Nowak & Sigmund (1998), *Nature*, 393, 573-577

### 4.3 强互惠与利他惩罚

```
U_i = material_i - c_punish·I(punish) + β·(norm_enforcement - cost_to_self)
惩罚决策：violation_severity > personal_threshold - social_pressure
```

**论文**：Fehr & Gachter (2002), Bowles & Gintis (2004)

## 五、地位与领导力（5 个模型）

### 5.1 威望 vs 支配双通道（Henrich & Gil-White, 2001）

```
Prestige_i(t+1) = Prestige_i(t) + α·Σw_ji·Skill_j·Deference_ji - decay·Prestige_i(t)
Dominance_i(t+1) = Dominance_i(t) + β·Σw_ij·(1-Resistance_j)·Coercion_ij - decay·Dominance_i(t)
Overall_Status = γ·Prestige + (1-γ)·Dominance
```

### 5.2 马太效应（Merton, 1968）

```
dS_i/dt = a·S_i + b·contribution_i
高阶地位角色的贡献获得的地位增益更大
```

### 5.3 期望状态理论（Berger, 1977）

```
E_ij = Σ_k w_k · (state_ik - state_jk) · τ_k
Status_i = (1/n) · Σ_j Conferral_ji
Conferral_ji = f(Perceived_Competence, Perceived_Generosity)
```

### 5.4 Hollander 信用模型（1958）

```
IC_i(t) = Σ [Conformity·w_c + Competence·w_comp - Deviance·w_d] · γ^(t-τ)
```

**适用场景**：领袖"攒人品"——早期循规蹈矩+展现能力=攒信用，以后才能做有争议的决策。

### 5.5 Keltner 权力腐败模型（2003, 2016）

```
Power ↑ → Empathy ↓, Overconfidence ↑
Abuse_Likelihood = σ(θ₁·Power + θ₂·Lack_of_Accountability - θ₃·Institutional_Constraints)
```

## 六、谣言与信息传播（4 个模型）

### 6.1 DK 谣言模型（Daley & Kendall, 1964）

```
dI/dt = -β·I·S / N
dS/dt = β·I·S / N - α·S·(S+R) / N
dR/dt = α·S·(S+R) / N
```

三态：Ignorant(未听说) → Spreader(传播) → Stifler(停止)

### 6.2 谣言变异（Allport & Postman, 1947）

```
M_{t+1} = M_t · (1 - λ) + δ_t + ε_t
λ = leveling（细节丢失），δ = sharpening（剩余放大），ε = assimilation（偏向同化）
```

### 6.3 八卦与社会凝聚力（Dunbar, 1992）

```
最大稳定群体规模：N_max = 150（邓巴数）
65% 对话涉及社交话题
需维持至少 42% 关系密度以保持凝聚
```

### 6.4 恐慌级联（Granovetter, 1978）

```
P(panic_i) = σ(α_i · (visible_panic_proportion - T_i))
T_i 为角色 i 的恐慌阈值，阈值分布决定级联是否发生
```

## 七、集体记忆（2 个模型）

### 7.1 Ebbinghaus 遗忘曲线

```
R(t) = exp(-t / S)
S = 记忆稳定性（随复述增加）
```

### 7.2 社会性检索诱导遗忘

```
R_i(t+1) = R_i(t) + μ·(mentioned - R_i(t))  // 被提到的细节强化
R_i(t+1) = R_i(t) - ν·R_i(t)                 // 被省略的抑制

5 个人对同一件事 → 记忆逐渐趋同（但不一定准确）
```

**论文**：Coman et al. (2016), *Nature Communications*, 7, 10830

## 八、群体认同（2 个模型）

### 8.1 社会认同形成（Tajfel & Turner, 1979）

```
ID_i = α·ingroup_density + β·outgroup_salience + γ·common_fate
分配偏见：Allocation_in = base + bias·ID, Allocation_out = base - bias·ID
```

**最小群体范式**：仅仅分类就足以产生内群偏好。

### 8.2 谢林隔离模型（1971）

```
当同类型邻居比例 < B_a（通常约 1/3）时移动
即使每个人只要求 35% 同类邻居，宏观结果也是极端隔离
```

## 九、社会网络动力学（3 个模型）

### 9.1 三边闭包

```
A 信任 B 且 A 信任 C → B↔C 每天 +0.5（朋友的朋友是朋友）
```

### 9.2 平衡理论

```
A 敌 B 且 B 敌 C → A↔C 每天 +0.3（敌人的敌人是朋友）
A 敌 B 且 B 友 C → A↔C 每天 -0.3（朋友的敌人是敌人）
```

### 9.3 邓巴等级（关系饱和度）

```
第 n 个关系的社交价值 = base / n^0.6
亲密 5 人、亲密 15 人、朋友 50 人、熟人 150 人
超过上限后新关系的价值趋近于零
```

## 十、集体行为（4 个模型）

### 10.1 阈值级联模型（Granovetter, 1978）

```
P(join) = 1 if fraction_joined ≥ threshold_i
最终参与比例 P 满足：P = F(P)（自洽方程）
阈值分布决定了级联规模：从 0% 到 100% 皆可能
```

### 10.2 复杂传染（Centola & Macy, 2007）

```
P(adopt) = σ(exposure_count - threshold)
简单传染：一次接触 → 采纳
复杂传染：需多次接触 → 高风险行为的传播需要社会证明
```

**论文**：Centola & Macy (2007), *AJS*, 113(3), 702-734

### 10.3 信息级联（BHW, 1992）

```
接受者 - 拒绝者 ≥ 2 → 级联锁定
之后所有人忽略私人信号
```

### 10.4 阿施从众（1951）

```
P(conformity) = majority_size / (majority_size + minority_size + 1)
3-4 人的多数派已达最大效果
高不确定性 → 从众概率 ↑
公开场合 → 从众概率 ↑
```

## 十一、信任与关系演化（3 个模型）

### 11.1 信任更新

```
Δtrust = +reward_weight·(benefit - expected)  // 合作
       = -penalty_weight·(expected - actual)    // 背叛
不对称性：|penalty_weight| > |reward_weight|（负面偏见约 2-3 倍）
修复一次背叛需要 2-3 次合作的信任增益
```

### 11.2 关系衰减

```
trust -= 0.5·decay_rate(loyalty)  // 3 天无互动后开始衰减
familiarity -= 1.0·decay_rate(loyalty)
decay_rate(loyalty) = 1.0 - loyalty·0.5  // 忠诚的人衰减慢
```

### 11.3 关系标签自动升级/降级

```
信任≥8 熟悉≥9 → "挚友"
信任≥6 熟悉≥7 → "朋友"
信任≥4 熟悉≥5 → "熟人"
信任<0 → "有敌意"
信任≤-6 → "敌人"
```
