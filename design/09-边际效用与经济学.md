## 一、边际效用递减（4 个模型）

### 1.1 三种效用函数

| 函数 | 公式 | 特性 | 适用 |
|------|------|------|------|
| 对数型 | U(x) = ln(x+1) | 强饱足感 | 生存必需品（食物、水） |
| 幂函数 | U(x) = x^α, 0<α<1 | 经典递减 | 一般商品（燃料、木材） |
| 指数饱和 | U(x) = 1 - e^(-kx) | 有上限 | 社交变量（信任、地位） |

```rust
pub enum UtilityCurve {
    Logarithmic,
    Power { alpha: f64 },      // α ≈ 0.3-0.8
    ExponentialSaturation { k: f64 },
}

impl UtilityCurve {
    pub fn evaluate(&self, x: f64) -> f64 {
        match self {
            Self::Logarithmic => if x <= 0.0 { 0.0 } else { (x + 1.0).ln() },
            Self::Power { alpha } => if x <= 0.0 { 0.0 } else { x.powf(*alpha) },
            Self::ExponentialSaturation { k } => 1.0 - (-k * x).exp(),
        }
    }

    /// 边际效用 U'(x)
    pub fn marginal(&self, x: f64) -> f64 {
        let eps = 1e-6;
        (self.evaluate(x + eps) - self.evaluate(x)) / eps
    }
}
```

### 1.2 边际效用与激励

```
边际效用 = U'(x)
当边际效用 < 阈值时，角色停止获取（满足化）
角色之间的阈值差异由人格决定：
  高神经质：不会停止囤积（"永远不够"）
  低神经质：满足于"差不多就行"
```

### 1.3 社交边际效用

```
第 n 个关系的效用 = base / n^0.6（邓巴等级）
第 n 个团队成员的生产力 = n^0.3（厨房太挤效应）
```

### 1.4 边际效用提示词注入

```rust
fn build_marginal_utility_prompt(char: &Character, resource: &str) -> String {
    let mu = char.utility_curve.marginal(char.resources.get(resource));
    format!(
        "{} 的边际效用是 {:.2}（{}）",
        resource, mu,
        if mu > 0.5 { "低——你还有很多" } else { "高——快用完了" }
    )
}
```

## 二、前景理论——损失厌恶（3 个模型）

### 2.1 Kahneman-Tversky 估值函数

**公式**
```
U(gain) = x^0.88           // 收益：递减愉悦
U(loss) = -2.25 · x^0.88   // 损失：2.25 倍的痛苦
```

损失比收益痛 2.25 倍。这意味着：
- 失去一个罐头的感觉 ≈ 发现两个罐头的快感
- 角色更愿意冒险避免损失（而不是获取收益）
- 损失造成的情绪影响比等量收益持续更久

### 2.2 参考点依赖

```
角色不按绝对水平评估结果，而是按相对于参考点的变化评估。

参考点 = 历史水平的指数移动平均
适应率：每日向当前水平移动 5-10%
```

**趋势比水平更重要**
```
心情好坏 ≠ 有多少物资
心情好坏 = 物资趋势（正/负/稳定）
角色在一个稳定的 80 分状态和高开低走的 50 分之间，前者更幸福
```

### 2.3 享乐跑步机

```
任何消费水平的愉悦感都会在 3-7 天内回归基线
发现的物资→兴奋 3 天→基线
失去的物资→痛苦 7-30 天→部分恢复（但不会回到 0）
```

**论文**：Kahneman & Tversky (1979), *Econometrica*, 47(2), 263-291; Brickman & Campbell (1971)

## 三、满意化决策（3 个模型）

### 3.1 满意化原则（Simon, 1956）

```
不优化，只满意。
每种需求有一个"足够好"阈值：
  饥饿 < 30：停止找食物
  信任 > 6：不再积极建立该关系
  安全 > 7：不优先考虑防御

只有当环境变化超过阈值时才打破满意状态
```

### 3.2 满意化 vs 优化

| 特质 | 满意阈值 | 行为 |
|------|---------|------|
| 高神经质 | 高（永不知足） | 持续囤积、焦虑 |
| 低神经质 | 低（容易满足） | 够用就行、放松 |
| 高尽责性 | 精确、依赖目标 | 有计划但灵活 |
| 低尽责性 | 模糊、随意 | 差不多就行 |

### 3.3 满意化状态切换

```
角色在大部分时间处于满意化状态（重复习惯行为，不调 LLM）
当环境剧烈变化时 → 打破满意化 → 触发 LLM 决策
```

**论文**：Simon (1956), *Psychol. Rev.*, 63(2), 129-138

## 四、机会成本（1 个模型）

**公式**
```
机会成本 = max(次优方案的价值 - 选定方案的价值, 0)
每个 tick 角色只能做一个主要动作
选择 A 意味着不选择 B
```

```rust
pub struct OpportunityCost {
    pub chosen_value: f64,
    pub foregone_value: f64,
    pub cost: f64,
    pub narrative_weight: f64,
}

impl OpportunityCost {
    pub fn between(chosen: f64, foregone: f64) -> Self {
        let cost = (foregone - chosen).max(0.0);
        Self { chosen, foregone, cost, narrative_weight: (cost / chosen.max(0.01)).min(1.0) }
    }
}
```

## 五、时间贴现（2 个模型）

### 5.1 指数贴现（理性但错误）

```
PV = R / (1 + δ)^t
时间一致性偏好
```

### 5.2 双曲贴现（人类真实）

```
PV = R / (1 + k·t)
时间不一致性——今天看要做的事，明天到了就不想做了
```

**关键差异**：双曲贴现产生拖延和后悔——角色计划"明天去搜索"，但明天到了选择再等一天。

**论文**：Ainslie (1975), *Psychol. Bull.*; Laibson (1997), *QJE*

## 六、需求弹性（3 个模型）

### 6.1 自身价格弹性

```
ε = (ΔQ/Q) / (ΔP/P)

|ε| < 1：缺乏弹性（必需品——无论如何都要买）
|ε| > 1：富有弹性（奢侈品——涨价就放弃）
```

**末日商品弹性**

| 商品 | 弹性 | 含义 |
|------|------|------|
| 食物 | -0.3 | 必需品，必须获取 |
| 药品 | -0.2 | 高度必需品 |
| 烟草 | -1.8 | 奢侈品，可放弃 |
| 燃料 | -0.5 | 必需品，但可替代 |

### 6.2 交叉弹性

```
ε_AB = (ΔQ_A/Q_A) / (ΔP_B/P_B)
ε > 0：替代品（燃料↑→绝缘材料需求↑）
ε < 0：互补品（燃料↑→车辆使用↓）
```

### 6.3 吉芬商品与凡勃伦商品

**吉芬商品**（稀缺加剧时需求反而上升）：
```
在极度贫困下：主食涨价→买更多主食（买不起替代品了）
→ 正向反馈循环：涨价→囤积→更短缺→更涨价
```

**凡勃伦商品**（越贵越想要）：
```
价格上升→地位信号增强→需求上升
仅对奢侈品有效
```

**论文**：Marshall (1890), *Principles of Economics*; Jensen & Miller (2008), *AER*

## 七、生产函数（2 个模型）

### 7.1 柯布-道格拉斯生产函数

```
Q = L^α · K^β
其中 α+β < 1 → 规模报酬递减
```

| 任务 | α（劳动力） | β（资本） | 最优人数 |
|------|-----------|---------|---------|
| 觅食 | 0.6 | 0.2 | 3-4 |
| 建筑 | 0.7 | 0.3 | 5-7 |
| 侦察 | 0.4 | 0.1 | 2-3 |

### 7.2 边际劳动产出

```
MP(L) = Q(L) - Q(L-1)
当 MP < 边际成本时，停止加人
```

**适用场景**：9 个人觅食的产出不是 1 个人的 9 倍，而是约 3.8 倍。3 人小队最高效。

**论文**：Cobb & Douglas (1928), *AER*

## 八、易货贸易（2 个模型）

### 8.1 交换比率确定（纳什讨价还价解）

```
rate_{A→B} = (urgency_B/urgency_A) · (scarcity_B/scarcity_A) · (1+power_ratio)
```

### 8.2 双重巧合问题

```
直接交换概率 = P(A有X且要Y) × P(B有Y且要X)
引入中间人后效率指数级提升
```

## 九、公共池资源与保险（3 个模型）

### 9.1 Ostrom 公地管理 8 原则

| # | 原则 | 评分 |
|---|------|------|
| 1 | 明确的边界 | 1 - outsider_use_ratio |
| 2 | 规则与当地条件匹配 | 1 - violation_rate |
| 3 | 集体选择安排 | 投票权成员 / 总成员 |
| 4 | 监督 | 1 - uncaught_violations / total |
| 5 | 分级制裁 | 警告→减配→强制劳动→驱逐 |
| 6 | 冲突解决机制 | 1 - unresolved/conflicts |
| 7 | 对组织权的认可 | 0/1 外部是否承认 |
| 8 | 嵌套式企业 | 有效治理层级数 |

综合可持续性评分 = Σ(原则_i × w_i)。< 阈值时公地崩塌。

**论文**：Ostrom (1990), *Governing the Commons*

### 9.2 保险池稳定性

```
expected_payouts = Σ p_claim_i · claim_amount_i
崩溃条件：连续多次索赔耗尽储备 或 成员退出 < 阈值 或 道德风险
```

### 9.3 合作相变模型

```
C(S) = 1 / (1 + e^{k(S - S_critical)})
S_critical ≈ 0.7-0.8，超过此阈值合作崩溃
```

**小说阶段判断**
```
五金店 3-10人：S≈0.5 → 合作度高
学校建设期 14-85人：S≈0.3 → 合作峰值
学校饱和 220人：S≈0.6-0.7 → 告密/私藏/派系分化
南迁前夜：S≈0.8 → 接近临界，不搬家就崩
```
