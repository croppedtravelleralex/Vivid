### 3.4 角色 API 映射

| 操作 | API | 组件 |
|------|-----|------|
| 角色列表 | `GET /api/v1/characters` | `CharacterList.vue` |
| 角色详情 | `GET /api/v1/characters/:id` | `CharacterDetail.vue` |
| 新建角色（手动） | `POST /api/v1/characters` | `CharacterCreate.vue` |
| LLM 生成 | `POST /api/v1/characters/generate` | `CharacterGenerate.vue` |
| LLM 批量生成 | `POST /api/v1/characters/batch-generate` | `CharacterBatchGen.vue` |
| 更新角色 | `PUT /api/v1/characters/:id` | `CharacterEdit.vue` |
| 部分更新 | `PATCH /api/v1/characters/:id` | (任何编辑操作) |
| 删除角色 | `DELETE /api/v1/characters/:id` | 删除确认对话框 |
| 修改状态 | `PATCH /api/v1/characters/:id/state` | GM 面板 |
| 修改位置 | `PATCH /api/v1/characters/:id/state`（location） | GM 面板传送 |

### 4.2 交互清单

| 交互 | 行为 | 实现 |
|------|------|------|
| **拖拽节点** | 手动调整布局，释放后力模拟重新收敛 | d3.drag() + simulation.reheat() |
| **悬停节点** | 高亮该节点及其直连边，其他 0.2 透明度 | CSS filter + opacity 过渡 |
| **单击节点** | 右侧面板显示角色摘要 | 切换选中状态，显示 CharSummaryPanel |
| **双击节点** | 跳转到角色详情页 | router.push(`/character/${id}`) |
| **滚轮缩放** | 缩放 0.2x ~ 5x，保持中心点 | d3.zoom() |
| **拖拽边** | 调整弯曲度（仅视觉） | d3 drag + path interpolation |
| **点击边** | 右侧面板显示关系编辑表单 | EdgeEditPanel |
| **右键空白** | 上下文菜单：重置视图/导出/全屏 | contextmenu event |
| **双击空白** | 添加新节点（新建角色入口） | 弹出角色 QuickCreate |
| **Ctrl+F** | 搜索角色并自动定位 | 搜索框 + 高亮 + 动画聚焦 |
| **拖拽文件到图** | 导入角色卡 JSON | 文件解析 + POST API |
| **选中多节点** | Shift+点击→批量操作菜单 | 多选状态 + context menu |

  "type": "relationship_changed",
  "data": {
    "charA": "zhangyuan",
    "charB": "linshuang",
    "delta": { "trust": "+1", "familiarity": "+1" },
    "newValues": { "trust": 7, "familiarity": 9 },
    "reason": "林霜帮张远加固了院墙"
  }
}

// 前端处理：重新加热力模拟
function onRelationshipChanged(data: RelationshipChange) {
  // 更新本地 edge 数据
  const edge = graphData.edges.find(
    e => (e.source.id === data.charA && e.target.id === data.charB)
  );
  if (edge) {
    edge.trust = data.newValues.trust;
    edge.familiarity = data.newValues.familiarity;
  }
  // 重新加热力模拟（小的扰动，快速收敛）
  simulation.alpha(0.3).restart();
}

## 十、实体删除工作流

删除任何实体（角色/地点/阵营/事件）都遵循**安全删除**流程：

```vue
<template>
  <div class="modal" v-if="show">
    <div class="modal-content delete-confirm">
      <h3>删除{{ entityType }}</h3>
      <div class="warning" v-if="warnings.length">
        <p>⚠️ 删除后可能产生以下问题：</p>
        <ul>
          <li v-for="w in warnings" :key="w">{{ w }}</li>
        </ul>
      </div>
      <div class="options">
        <label>
          <input type="radio" v-model="mode" value="hard" />
          硬删除：从世界状态中彻底移除
        </label>
        <label>
          <input type="radio" v-model="mode" value="mark_dead" />
          标记死亡：状态设为"dead"，保留历史和关系
        </label>
        <label>
          <input type="radio" v-model="mode" value="archive" />
          归档：保留数据但不再参与模拟
        </label>
      </div>
      <div class="cascade-options" v-if="entityType === 'character'">
        <label>
          <input type="checkbox" v-model="generateDeathEvent" />
          生成"死亡"事件（触发相关 hook 和情绪反应）
        </label>
        <label>
          <input type="checkbox" v-model="reassignRelationships" />
          自动重分配关系（好友转给指定角色）
        </label>
      </div>
      <div class="actions">
        <button @click="confirmDelete" class="danger">{{ confirmText }}</button>
        <button @click="$emit('close')">取消</button>
      </div>
    </div>
  </div>
</template>

<script setup>
const props = defineProps({
  entityType: String,   // 'character' | 'location' | 'faction' | 'event'
  entityId: String,
  entityName: String,
});

async function confirmDelete() {
  await api.delete(`/api/v1/${entityType}s/${props.entityId}`, {
    data: {
      mode: mode.value,  // 'hard' | 'mark_dead' | 'archive'
      generateEvent: generateDeathEvent.value,
      reassignTo: reassignTarget.value,
    }
  });
  emit('deleted');
  emit('close');
}
</script>
```

## 十一、WebSocket 实时同步清单

所有页面通过 WebSocket 接收增量更新，不轮询：

| WsEvent type | 触发条件 | 影响页面 | 处理动作 |
|------|--------|---------|---------|
| `tick` | 每 tick | 全部 | 更新时间/tick/速度 |
| `character_update` | 角色状态变化 | Dashboard, CharacterList, CharacterDetail | 局部更新 role 数据 |
| `relationship_changed` | 关系变化 | 关系图, CharacterDetail | 更新边数据，重新加热 |
| `event_triggered` | 新事件 | Dashboard, Timeline | 事件流前置插入新条目 |
| `environment_update` | 环境变化 | Dashboard, World | 更新环境卡片数据 |
| `speed_changed` | 速度切换 | Header, TimeControl | 更新速度按钮状态 |
| `council_completed` | 议会完成 | 议会面板 | 弹窗提示 + 刷新议会内容 |
| `tag_updated` | 标签/线索变化 | 标签面板 | 更新线索列表 |
| `character_added` | 角色新建 | CharacterList, 关系图 | 添加新节点 |
| `character_removed` | 角色删除 | CharacterList, 关系图 | 淡出移除节点 |
| `location_changed` | 地点变化 | World, 地点面板 | 刷新地点数据 |
| `log` | 引擎日志 | StatusBar 控制台 | 追加日志条目 |
